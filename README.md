# Watcher

Watcher is a simple file status monitor service

## usage 

On Linux

```shell script
ihexon@gentoo$ chmod +x watcher
ihexon@gentoo$ ./watcher -r $DIR
```

Other Platform

```shell script
java -jar Watcher.jar -r $PATH
```
